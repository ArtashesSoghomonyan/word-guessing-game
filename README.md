# Word Guessing Game

This is a terminal version of classic word guessing game written in python.

## How to start the game

#### 1. Clone this repository using git

```shell
  $ git clone https://github.com/ArtashesSoghomonyan/word-guessing-game
```

#### 2. Go to the cloned repo folder

```shell
  $ cd word-guessing-game
```

#### 3. Start a python virtual environment and activate it

``` shell
  $ python3 -m venv env
  $ source env/bin/activate
```

#### 4. Install requirements

``` shell
  $ pip install -r requirements.txt
```

#### 5. Now it's ready to run!

``` shell
  $ python3 main.py
```
